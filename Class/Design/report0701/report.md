# メモ
2変数程度のものがいい
講義の例は、一定の体積をもつ円筒の表面積を最小化する問題
目的関数：表面積
設計変数：半径、高さ
制約関数：体積

同じような幾何学的な問題を考える
例：
- 一定の体積をもつ直方体の表面積を最小化する問題
- 一定の体積をもつ球の表面積を最小化する問題
- 一定の体積をもつ正多面体の表面積を最小化する問題
- 一定の体積をもつ楕円体の表面積を最小化する問題
- 一定の体積をもつ円錐の表面積を最小化する問題
